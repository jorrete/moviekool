# MovieKOOL

Dummy streaming platform.

## Development

```bash
git clone https://github.com/jorrete/moviekool.git
cd moviekool
npm install
npm run dev
```

## Testing

Run tests on time.

```bash
npm run test:run
```

Run tests in **watch** mode.

```bash
npm run test:watch
```

Run tests in **watch** mode with a browser UI.

```bash
npm run test:UI
```

## Deploying

Build a check the result. It will check:

- stylelint
- eslint
- typescript

```bash
npm run preview
```
